서비스 결제

해지 및 환불 안내

유료서비스 신청 후 해지하실 경우 해지수수료를 제외한 잔여분을 환불해드립니다.
해지수수료는 가입일부터 적용됩니다.

환불액 계산

부가세를 제외한 결제금액 - { (1일 이용요금) x (이용일수) x ( 해지수수료) }

1일 이용요금 1개월 : 30일
3개월 : 90일
해지수수료 서비스 가입요금의 10%

환불액 계산

99만원 결제시 부가세 9만원을 제하고
90만원 - (3만원x7일) + 해지수수료 : 9만원)=60만원

※ 카드 결제하신 경우, 결제금액은 정상 청구되고, 환불금은 해지시에 알려주신 은행 계좌로 현금으로 환불됩니다.

※ 7일 미만 사용시에도 서비스제공 기본일수 7일로 처리됩니다.
